# QA 보고서: 책 이름 자동 짧게 표시

- 일시: 2026-05-27
- 범위: 책 목록 버튼과 장 보기 헤더에서 NT(신약) 책 이름을 화면 폭에 맞춰
  자동으로 줄여 보여주는 동작
- 파일:
  - 유닛: `tests/unit/views-routing.test.js` (기존 케이스 무회귀, 537/537 통과)
  - e2e: `tests/e2e/test_book_name_swap.py` (7 케이스 신규)

## 무엇을 확인했나

신약 책 이름은 길어서(예: "고린토인들에게 보낸 첫째 편지") 화면이 좁거나
글자가 크면 버튼·헤더가 2줄로 부풀어 가독성이 떨어졌다. 이번 작업은 두
곳에서 자동으로 짧은 이름으로 바꾸어 보여준다:

- 책 목록(`/`, `/old_testament`, `/new_testament`, `/deuterocanon`) 버튼
- 장을 읽을 때 상단 헤더("책 이름 + N장")와 장 목록·머리말 헤더

복음서 4권과 사도행전은 원래 이름이 짧아 줄이지 않는다.

## 검증한 영역

### 1. 책 목록 버튼

| 화면                                 | 표시                                                  |
| ------------------------------------ | ----------------------------------------------------- |
| 아이폰·안드로이드폰 (터치 기기)      | 항상 짧은 이름 (예: "Ⅰ고린토")                              |
| 아이패드·태블릿 (터치 기기)          | 가로모드처럼 넓어도 짧은 이름                               |
| 데스크탑/노트북 (마우스) — 기본 글자 | 좁은 버튼이면 자동으로 짧은 이름, 여백 있으면 정식 명칭     |
| 데스크탑 — 글자 키움                 | 2줄이 될 만하면 자동으로 짧은 이름                          |

핵심 동작: 터치 기기는 무조건 짧은 이름, 비-터치는 "버튼이 2줄이 될 만하면"
짧은 이름. 글자 크기를 키우면(브라우저 줌, OS 설정, 앱 내 글자 크기) 자동으로
다시 측정해서 결정한다.

### 2. 장 보기 헤더 (책 이름 + 장/편)

| 화면                        | 표시                                |
| --------------------------- | ----------------------------------- |
| 아이폰 (390px 폭)           | "Ⅰ고린토 5장"                       |
| 아이패드 세로 (768px)       | "고린토인들에게 보낸 첫째 편지 5장" |
| 데스크탑 (1280px) 기본 글자 | 정식 명칭                           |
| 데스크탑 + 큰 글자(32px)    | "Ⅰ고린토 5장" — 자동으로 짧은 이름  |

핵심: 좌우의 ← 뒤로가기 / 🔖 북마크 버튼이 차지하는 자리를 빼고도 제목이
한 줄에 들어가는지 매번 측정한다. 안 들어가면 짧은 이름으로 자동 교체.

### 3. 정식 명칭은 스크린리더가 항상 듣는다

화면에 짧은 이름("Ⅰ고린토")이 보여도 시각 장애인용 스크린리더는 정식
명칭("고린토인들에게 보낸 첫째 편지 5장")을 읽는다. `aria-label`이 항상
정식 명칭을 가리키도록 처리했다.

### 4. 복음서 4권 + 사도행전은 그대로

이미 이름이 짧아(예: "마태오의 복음서") 짧게 만들지 않는다. e2e 가 이 점을
명시적으로 확인 — 마태복음 페이지에서는 짧은 이름용 span 자체를 만들지 않고
정식 명칭만 표시.

## 무엇을 확인 안 했나 (의도적 범위 제외)

- **iOS/Safari 실기기 동작** — Playwright는 Chromium 기반이라 Safari 고유의
  글자 렌더링 차이는 보지 않는다. iPhone Safari 실제 화면은 개발자가 수동
  확인할 것
- **외경/구약 책 이름** — 본 작업 범위 밖. 사용자 사양상 신약만 짧은 이름
  매핑을 제공. 외경/구약의 긴 이름은 화면 폭이 충분하면 정식 명칭 그대로,
  공간이 부족해 2줄이 되어도 짧은 이름 매핑이 없으므로 줄어들지 않음
- **글자 크기 변경 후 시각 깜빡임** — 측정·교체가 매우 빠르지만 한 프레임
  정도의 깜빡임은 발생할 수 있다 (브라우저 zoom·앱 글자 크기 변경 직후).
  실사용에서는 거의 인지 불가
- **다른 곳의 책 이름 표시** — 검색 결과, 북마크 목록 등은 별도 모듈
  (`bookmark.js` / `search.js`)에 자체 로직(`short_name_ko`)이 있어 본
  작업과 분리된 정책을 따른다
