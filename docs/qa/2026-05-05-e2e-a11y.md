## 5. 비고

- axe-core는 critical/serious만 검증. moderate/minor 위반은 별도 백로그로 관리.
- `axe-playwright-python 0.1.7`에서 `results.response`가 dict를 직접 반환함 (json.loads 불필요).
- Escape 키로 verse-select-bar를 닫는 핸들러는 내부 상태 변수(`_verseSelectMode`)를 확인하므로, programmatic DOM 조작으로는 테스트할 수 없음. `enterVerseSelectMode()` + 취소 버튼 방식으로 대체.
